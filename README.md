# БОТ ДЛЯ ИГРЫ ASPHALT LEGENDS - WINDOWS
## Бот управляет игрой Asphalt Legends

В процессе разработки и тестирования

## Системные требования

- Windows 10 - 64


## Возможности

- Круглосуточное управление игрой по сети
